# 자기소개 페이지

`cursorstudy` 폴더의 자기소개 페이지 구성과 실행 방법을 정리합니다.

## 파일 구성

- `index.html`: 메인 페이지
- `style.css`: 페이지 스타일
- `script.js`: 네비게이션 하이라이트 및 강조 효과
- `자기소개서 실습용.txt`: 원본 자기소개 내용

## 실행 방법

1. `index.html`을 브라우저에서 열어 확인합니다.
2. 스타일과 스크립트가 정상적으로 연결되었는지 확인합니다.

## 커스터마이징

- 색상/폰트: `style.css`의 `:root` 변수 수정
- 문구: `index.html`의 섹션 텍스트 수정
- 동작: `script.js`의 이벤트 로직 수정
